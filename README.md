# 📖 安装指南

## Claude Code

```bash
# 克隆仓库
git clone https://github.com/your-username/best-skills-research-writing.git

# 复制 skills 到 Claude Code 目录
cp -r best-skills-research-writing/skills/* ~/.claude/skills/
```
